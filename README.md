SuperStore Sales Data Analysis – Power BI Dashboard Project

Overview
This project focuses on analyzing retail sales data using Power BI to uncover business insights related to sales performance, profitability, customer segments, and regional trends. The goal is to demonstrate practical data analytics skills including data cleaning, visualization, and business reporting.

Dataset
The dataset used in this project is the SuperStore Sales Dataset in CSV format. It contains detailed records of customer orders including order date, region, product category, sales, profit, discounts, and customer segments.

File used:
SuperStore_Sales_Dataset.csv

Tools & Technologies
Power BI – Data modeling, visualization, DAX calculations
Microsoft Excel / CSV – Raw dataset handling
Windows OS

Project Workflow

Imported the CSV dataset into Power BI

Performed data cleaning and transformation using Power Query

Created relationships and calculated columns/measures using DAX

Designed an interactive sales dashboard

Analyzed sales, profit, customer behavior, and regional performance

Generated business insights for decision-making

Dashboard Features
• Total Sales, Profit, and Order Count KPIs
• Sales and Profit by Region
• Category and Sub-Category Performance
• Customer Segment Analysis
• Monthly and Yearly Sales Trends
• Top Performing Products

Key Insights
• Identified the most profitable product categories
• Analyzed which regions generate the highest revenue
• Found high-discount areas affecting profitability
• Compared customer segments based on revenue contribution

How to Use the Project

Download the repository

Open the Power BI file (Dashboard_.pbix) using Power BI Desktop

Make sure the CSV file is in the same folder if refresh is required

Explore the interactive dashboard and filters
